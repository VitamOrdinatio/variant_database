# Schema Philosophy

```text
Metadata Layer
    What exists?

Provenance Layer
    How did it come to exist?

Data Layer
    What biological substrate is preserved?

Relational Layer
    How is it persisted without collapse?

Discovery Layer
    How does VDB expand its evidence horizon?

Reasoning Surface Layer
    How do downstream consumers safely reason over preserved evidence?
```
