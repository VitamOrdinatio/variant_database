# Validation Results

This directory contains git-tracked validation receipts for VDB.

Validation receipts are generated evidence from validation or certification runs. They are retained when they support governance, reproducibility, benchmark evidence, certification evidence, or historical validation review.

This directory is not for disposable runtime scratch output.
