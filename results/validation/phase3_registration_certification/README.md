This directory contains the generated artifacts supporting the Phase 3 Registration Certification documented in `docs/validation/phase3_registration_certification.md`. 

These files are generated directly by the registration efficacy probe (`scripts/mark/probe_mark_phase3_registration_efficacy.py`) and constitute the primary evidence supporting the certification conclusions.

For executive summary and analysis of Phase 3 Registration Certification, see the offical [Phase3 Registration Certification](../../../docs/validation/phase3_registration_certification.md).
